# Инвентаризация содержания для модульной версии

## Назначение

Документ фиксирует, как существующие материалы `lesson01`, `lesson02` и
`lesson03` будут использоваться в семи модулях. На этой итерации исходные
уроки не изменяются и не перемещаются.

Обозначения решений:

- **перенести** — сохранить содержание с минимальной редактурой;
- **сократить** — оставить центральные задания, убрать повторения;
- **разделить** — распределить материал между базой, коррекцией и
  усложнением;
- **добавить** — в старых уроках нет необходимого элемента;
- **не включать** — содержание не входит в главный результат модуля.

## Общие решения по переносу

1. Входная диагностика старого урока не переносится целиком в каждый модуль:
   остаются только один-два вопроса на предварительные умения.
2. Самостоятельная проверка и выходной билет объединяются в одну контрольную
   точку; одинаковые задания удаляются.
3. Общая самооценка заменяется предметной.
4. Задания со звёздочкой не считаются обязательной частью базового маршрута.
5. Миллиарды сохраняются, но в M03–M04 маркируются как расширение, пока
   устойчиво не освоены тысячи и миллионы.
6. Учительская версия должна содержать решение по маршруту, а не только
   правильные ответы.
