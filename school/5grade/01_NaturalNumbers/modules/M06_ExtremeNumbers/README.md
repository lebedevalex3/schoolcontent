# M06. Наибольшее и наименьшее число

Шестой модуль раздела «Обозначение натуральных чисел» для 5 класса.

## Результат

Ученик:

- составляет наибольшее и наименьшее число из заданных цифр;
- использует каждую цифру ровно один раз;
- правильно располагает ноль при построении минимума;
- сравнивает варианты по первому различающемуся разряду;
- обнаруживает потерянные, лишние и неверно расположенные цифры;
- объясняет, почему построенное число является наибольшим или наименьшим.

## Предварительное условие

Ученик понимает значение позиции цифры, различает цифру и число, не ставит
ноль в начало записи многозначного числа и владеет упорядоченным перебором M05.

## Комплект

- `content-student.tex` — объяснение, практика, работа с ошибками и контроль;
- `content-teacher.tex` — ответы, диагностика и маршруты по результату;
- `content-correction.tex` — двухэтапная коррекционная карточка M06-R;
- `content-extension.tex` — поиск и обоснование второго по величине числа M06★;
- `content-homework-student.tex` — домашняя работа M06-H;
- `content-homework-teacher.tex` — ответы и критерии проверки M06-H;
- `main-*.tex` — сборочные файлы.

## Самостоятельный маршрут

1. Ученик выполняет основной лист и контрольную точку.
2. При ошибках с нулём, порядком или использованием цифр выполняет M06-R.
3. После M06-R проходит повторную проверку на новом наборе цифр.
4. Если способ понятен, но объяснение неустойчиво, выполняет M06-H или
   повторяет задания на первый различающийся разряд.
5. При уверенном результате переходит к M07; M06★ выполняется дополнительно.

Время выполнения не является критерием освоения. Переход определяется
правильностью чисел, использованием всех цифр и качеством разрядного
объяснения.

## Сборка

Из каталога модуля запустите `build-pdfs.sh` в macOS/Linux или
`build-pdfs.ps1` в Windows. Готовые цветные и чёрно-белые PDF сохраняются в
папке `outputs/` модуля.
