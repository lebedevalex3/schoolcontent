# M06. Наибольшее и наименьшее число

Шестой модуль раздела «Обозначение натуральных чисел».

## Результат

Ученик составляет наибольшее и наименьшее число из заданных цифр и
объясняет выбор через старший различающийся разряд.

## Предварительное условие

Ученик понимает значение позиции цифры, не ставит ноль в начало числа и
удерживает условие «каждая цифра используется ровно один раз» (M05).

## Ориентировочное время

25–40 минут. Базовый маршрут можно дополнить выбранными упражнениями.

## Комплект

- `content-student.tex` — базовый способ, практика и контроль;
- `content-teacher.tex` — ответы, диагностика и маршруты;
- `content-correction.tex` — пошаговый выбор M06-R;
- `content-extension.tex` — второе по величине число M06★;
- `main-*.tex` — сборочные файлы.

## Маршрут

`M05 → M06 → M07`. При ведущем нуле или выборе «на глаз» используется M06-R.
