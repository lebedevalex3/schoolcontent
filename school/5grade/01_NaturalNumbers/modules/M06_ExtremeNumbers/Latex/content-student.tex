\section{Сначала уточняем задачу}

\begin{notebox}[Два термина]
  \textbf{Наибольшее число} среди допустимых вариантов будем кратко называть
  \textbf{максимумом}, а \textbf{наименьшее число} \textemdash{} \textbf{минимумом}.
  Во всех заданиях каждую данную цифру нужно использовать ровно один раз.
\end{notebox}

\begin{fillbox}[Разобранный пример: как получить максимум]
  Из цифр \(2,\ 5,\ 7,\ 9\) составим наибольшее четырёхзначное число.
  На первое место ставим наибольшую цифру: \answerblank[12mm]. Затем каждый
  раз выбираем наибольшую из оставшихся. Получаем \answerblank[38mm].
\end{fillbox}

\begin{reflectionbox}[Правило и объяснение способа]
  При одинаковом количестве цифр числа сравнивают слева направо. Больше то
  число, у которого в \textbf{первом различающемся разряде} стоит большая
  цифра. Поэтому цифры максимума располагают по убыванию.
\end{reflectionbox}

\section{Минимум с нулём}

Из цифр \(0,\ 2,\ 5,\ 7\) нельзя получить четырёхзначное число \(0257\):
эта запись обозначает число 257. Поэтому простого расположения цифр по
возрастанию недостаточно.

\begin{fillbox}[Точный алгоритм]
  \begin{enumerate}[label=\textbf{\arabic*.}]
    \item На первое место поставьте наименьшую \textbf{ненулевую} цифру.
    \item Сразу после неё поставьте все нули.
    \item Остальные цифры расположите по возрастанию.
  \end{enumerate}
  Для \(0,\ 2,\ 5,\ 7\) получаем \answerblank[42mm].
\end{fillbox}

\begin{warningbox}[Почему не \(2507\)]
  Числа \(2057\) и \(2507\) впервые различаются в разряде сотен. Так как
  \(0<5\), то \(2057<2507\). Ноль ставим как можно левее, но не первым.
\end{warningbox}

\begin{practicebox}[9. Проверьте правила]
  Отметьте верные утверждения. Неверные исправьте.
  \begin{enumerate}[label=\choicebox]
    \item Для максимума цифры располагают по убыванию.

    \item При сравнении чисел с одинаковым количеством цифр ищут первый
