# M03. Классы многозначных чисел

Третий модуль раздела «Обозначение натуральных чисел».

## Результат

Ученик делит запись числа справа налево на классы и различает группу цифр,
которой записан класс, и число единиц этого класса.

## Предварительное условие

Пройдена контрольная точка M02: ученик сохраняет нулевые разряды при записи
числа.

## Ориентировочное время

Около 20 минут. Переход определяется контрольной точкой.

## Комплект

- `content-student.tex` — базовый лист ученика;
- `content-teacher.tex` — ответы и решения по маршруту;
- `content-correction.tex` — коррекционная карточка M03-R;
- `content-extension.tex` — усложнение M03★;
- `main-*.tex` — сборочные файлы.

## Итоговые PDF

- `M03-student-color.pdf` — 2 страницы;
- `M03-teacher-color.pdf` — 2 страницы;
- `M03-correction-color.pdf` — 1 страница;
- `M03-extension-color.pdf` — 1 страница.
