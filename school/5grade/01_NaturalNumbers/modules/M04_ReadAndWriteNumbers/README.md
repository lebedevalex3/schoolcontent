# M04. Чтение и запись многозначных чисел

Четвёртый модуль раздела «Обозначение натуральных чисел».

## Результат

Ученик читает многозначное число и записывает его цифрами, сохраняя нулевые
и неполные классы.

## Предварительное условие

Пройдена контрольная точка M03: ученик делит число на классы справа налево и
сохраняет три позиции в каждом непервом классе.

## Ориентировочное время

25–40 минут. Базовый маршрут можно завершить после контрольной точки, а блок
закрепления использовать сразу, позднее или выборочно.

## Комплект

- `content-student.tex` — лист ученика с базой и расширенной практикой;
- `content-teacher.tex` — ответы, диагностика ошибок и решения по маршруту;
- `content-correction.tex` — коррекционная карточка M04-R;
- `content-extension.tex` — усложнение M04★;
- `main-*.tex` — сборочные файлы.

## Маршрут

`M03 → M04 → M05`. Если ученик теряет нулевой или неполный класс, применяется
M04-R. Для уверенно выполнивших базу предназначен M04★.
