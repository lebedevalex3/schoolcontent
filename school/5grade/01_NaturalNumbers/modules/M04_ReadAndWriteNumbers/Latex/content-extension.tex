  Ученик получил запись $18\,40\,6$ и прочитал её: «восемнадцать миллионов
  четыреста шесть тысяч».

  \begin{enumerate}[label=\textbf{\arabic*.}]
    \item Какое число он, вероятно, хотел записать? \answerblank[55mm].
    \item Назовите ошибку в записи: \answerblank[80mm].
    \item Назовите отдельную ошибку в чтении: \answerblank[75mm].
    \item Запишите правильное чтение восстановленного числа:
      \answerblank[90mm].
  \end{enumerate}
\end{warningbox}

    \item есть один полностью нулевой класс;
    \item другой класс начинается с нуля;
    \item при чтении нулевой класс не произносится.
  \end{itemize}

  Чтение: \answerblank[110mm].

  Возможная ошибочная запись: \answerblank[65mm].

  Как по ошибке понять, какой класс потерян?

\end{practicebox}
