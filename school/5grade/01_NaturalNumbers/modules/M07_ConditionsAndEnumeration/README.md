# M07. Упорядоченный перебор и условия между цифрами

Заключительный модуль раздела «Обозначение натуральных чисел».

## Результат

Ученик переводит условие между цифрами в математическую запись, выбирает
удобную цифру для упорядоченного перебора, находит все подходящие числа и
обосновывает полноту проверкой всех допустимых значений.

Маршрут начинается с конкретной задачи о двузначном коде. Сюжетные задачи
используются выборочно для постановки цели и переноса способа; основная часть
практики остаётся краткой и математически прозрачной.

## Предварительные условия

Ученик понимает разряды, отличает цифру от её значения и умеет вести
упорядоченный перебор по первой цифре (M01, M05).

Модуль не ограничен временем. Переход определяется достижением результата в
контрольной точке, а объём практики выбирается по устойчивости способа.

## Комплект

- `content-student.tex` — модели условий, практика и контроль;
- `content-teacher.tex` — ответы, диагностика и маршруты;
- `content-correction.tex` — табличная коррекция M07-R;
- `content-extension.tex` — подсчёт без полного списка M07★: сначала
  конкретные условия и словесные закономерности, общие формулы только
  необязательно с учителем;
- `content-homework-student.tex` — домашняя работа M07-H;
- `content-homework-teacher.tex` — ответы и критерии проверки M07-H;
- `main-*.tex` — сборочные файлы.

## Сборка

Запускать из папки `Latex`:

```bash
xelatex -interaction=nonstopmode -halt-on-error main-student-color.tex
xelatex -interaction=nonstopmode -halt-on-error main-teacher-color.tex
xelatex -interaction=nonstopmode -halt-on-error main-correction-color.tex
xelatex -interaction=nonstopmode -halt-on-error main-extension-color.tex
xelatex -interaction=nonstopmode -halt-on-error main-homework-student-color.tex
xelatex -interaction=nonstopmode -halt-on-error main-homework-teacher-color.tex
```

Удобнее запускать `build-pdfs.sh` в macOS/Linux или `build-pdfs.ps1` в
Windows. Все цветные и чёрно-белые PDF сохраняются в папке `outputs/` модуля.

## Маршрут

`M06 → M07 → итоговая ревизия раздела`. В M07 ученик сначала выполняет
обязательное ядро и контроль; дополнительная практика назначается по
результату. При обращении зависимости или включении 10 как цифры используется
M07-R. Если способ понятен, но список остаётся неполным, используются
дополнительные задания или M07-H. После устойчивого контроля предлагается
M07★.
