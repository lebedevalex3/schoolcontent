\worksheettitle{Условия между цифрами}{\modulemark{M07} Версия учителя \textbullet\ маршрут 35–50 минут}

\begin{teacherbox}[Паспорт]
  \textbf{Результат:} ученик переводит условие в равенство, строит полный
  перебор и объясняет границу. \textbf{Опора:} M01 и M05.
\end{teacherbox}

\begin{checkpointbox}[Вход]
  (13,15,31,35,51,53). Полнота: проверены все три допустимые первые цифры и
  для каждой — две оставшиеся вторые цифры. Случайный список требует M05-R.
\end{checkpointbox}

\section{Базовая модель}
Пусть (a) — десятки, (b) — единицы. Для (b=a+2):
(13,24,35,46,57,68,79). При (a=8) получается (b=10), а 10 не цифра.

Для (b=2a): (12,24,36,48). Для (a+b=7):
(16,25,34,43,52,61,70).

\begin{teacherbox}[Три обязательных вопроса]
  Какая цифра изменяется по порядку? Как вычисляется вторая? Какое первое
  значение стало недопустимым и почему?
\end{teacherbox}

Равенства базы: (b=a+4); (a=b+3), или (b=a-3); (b=3a);
(a+b=9).

\section{Маршруты}
\begin{resultbox}[Дозирование]
  \begin{itemize}
    \item \textbf{Быстрый:} три модели, по одному заданию каждого типа,
      ошибки, контроль, M07\(\star\).
    \item \textbf{Основной:} вся база, 4 задания практики, смешанные условия,
      контроль.
    \item \textbf{С опорой:} только условия на больше/меньше, M07-R, затем
      кратность и сумма.
  \end{itemize}
\end{resultbox}

\section{Практика: ответы}
\begin{enumerate}
  \item (12,23,34,45,56,67,78,89) — 8;
  \item (30,41,52,63,74,85,96) — 7;
  \item (13,26,39) — 3;
  \item (21,42,63,84) — 4;
  \item (14,23,32,41,50) — 5;
  \item (19,28,37,46,55,64,73,82,91) — 9.
\end{enumerate}

\section{Ошибки и диагностика}
\begin{enumerate}
  \item Верно (b=a-2); числа (20,31,42,53,64,75,86,97).
  \item 10 не цифра; для (b=a+4): (15,26,37,48,59).
  \item Нужно включить (60): ноль допустим в единицах.
  \item Зависимость обращена. Для (a=2b): (21,42,63,84).
\end{enumerate}

Для (b=a+3) потеряны (47,58,69). Таблица продолжается до (a=6,b=9);
при (a=7) получается 10.

\begin{teacherbox}[Различайте причины]
  \begin{itemize}
    \item знак направлен неверно — обращение зависимости;
    \item появляется 10 — не удерживается понятие цифры;
    \item пропущено число с единицами 0 — ноль ошибочно исключён;
    \item верные ответы есть, но список короткий — нет упорядоченного перебора.
  \end{itemize}
\end{teacherbox}

\section{Смешанные условия}
\begin{enumerate}
  \item сумма 8 и (a>b): (53,62,71,80);
  \item (b=a+2) и число больше 40: (46,57,68,79);
  \item (a=2b) и число меньше 80: (21,42,63).
\end{enumerate}

Собственное условие оценивается по четырём признакам: точная фраза,
соответствующее равенство, полный список и объяснённая граница.

\section{Контроль и развилка}
Ответ: (b=a+3); (14,25,36,47,58,69). Последняя пара (a=6,b=9);
следующее значение (b=10) недопустимо.

\begin{resultbox}[Решение]
  \begin{itemize}
    \item \textbf{Раздел завершён:} равенство, список и граница верны.
    \item \textbf{M07-R:} зависимость обращена, включено 10 или список случаен.
    \item \textbf{M07\(\star\):} ученик готов считать допустимые строки без
      полного выписывания чисел.
  \end{itemize}
\end{resultbox}

После коррекции: «единицы на 4 больше десятков». Ожидается
(15,26,37,48,59) и остановка перед (6,10).
