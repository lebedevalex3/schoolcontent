# M05. Составление чисел из цифр

Пятый модуль раздела «Обозначение натуральных чисел».

## Результат

Ученик составляет полный упорядоченный список чисел из заданных цифр,
учитывая повторения, условие использования цифр и запрет ведущего нуля.

## Предварительные условия

Ученик различает цифру и число, понимает значение позиции цифры и умеет
читать полученные числа. Основная опора — результаты M01 и M02.

## Ориентировочное время

30–45 минут. Базовый маршрут завершается контрольной точкой; запас практики
используется выборочно по темпу класса.

## Комплект

- `content-student.tex` — база, расширенная практика и контроль;
- `content-teacher.tex` — ответы и решения по маршрутам;
- `content-correction.tex` — дерево выбора M05-R;
- `content-extension.tex` — проверка полноты M05★;
- `main-*.tex` — сборочные файлы.

## Маршрут

`M05 → M06`. При случайном переборе, самовольном запрете повторений или
ведущем нуле используется M05-R. Для уверенно выполнивших базу — M05★.
