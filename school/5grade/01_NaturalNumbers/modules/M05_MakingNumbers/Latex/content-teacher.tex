\worksheettitle
  {Составление чисел из цифр}
  {\modulemark{M05} Версия учителя}

\begin{teacherbox}[Паспорт модуля]
  \textbf{Результат:} ученик читает условие переборной задачи, строит полный
  упорядоченный список, учитывает повторения, обязательность цифр и запрет нуля
  в начале записи, обнаруживает пропуски и повторы.

  \textbf{Опора:} результаты M01 и M02. Формулы перестановок не вводятся.
  Время не ограничивается; переход определяется контрольной точкой.
\end{teacherbox}

\begin{teacherbox}[Структура ученического маршрута]
  Обязательное ядро включает вход, разобранные примеры, алгоритм, практику с
  опорой, самостоятельную практику, разбор ошибок и контроль. Задания на
  сравнение условий и составление условия по готовому списку обозначены как
  дополнительные.
\end{teacherbox}

\begin{checkpointbox}[Вход: решение]
  В числе \(525\) цифра десятков \(2\), её значение \(20\). Числа \(25\) и
  \(52\) различны, потому что цифры занимают разные разряды.
\end{checkpointbox}

\section{Читаем условие точно}

\begin{teacherbox}[Четыре параметра]
  Перед перебором ученик должен назвать: разрешённые цифры, длину числа,
  разрешение или запрет повторений, обязательность использования каждой цифры.
\end{teacherbox}

Для цифр \(2\) и \(5\):
\begin{itemize}
  \item двузначные, повторения разрешены, использовать обе цифры необязательно.
    \par Ответ: \(22,\ 25,\ 52,\ 55\);
  \item двузначные без повторений: \(25,\ 52\);
  \item трёхзначные с повторениями, обе цифры обязательны:
    \(225,\ 252,\ 255,\ 522,\ 525,\ 552\).
\end{itemize}

Дополнительное устное сравнение условий для цифр \(3\) и \(7\):
\begin{enumerate}[nosep]
  \item \(33,\ 37,\ 73,\ 77\);
  \item \(37,\ 73\);
  \item \(337,\ 373,\ 377,\ 733,\ 737,\ 773\).
\end{enumerate}

\begin{teacherbox}[Не допускайте двусмысленности]
  Фраза «использовать цифры 2 и 5» не задаёт точного условия. Уточняйте:
  можно ли брать другие цифры; обязательны ли цифры 2 и 5.
\end{teacherbox}

\section{Алгоритм и упорядочивание}

Из цифр \(2,\ 5,\ 7\), используя каждую ровно один раз:
\[
  257,\ 275;\qquad 527,\ 572;\qquad 725,\ 752.
\]

\begin{teacherbox}[Почему список полный]
  Проверены все три возможные первые цифры. После выбора первой цифры две
  оставшиеся располагаются двумя способами. Получено \(3\cdot2=6\) вариантов.
  Внутри каждой группы числа расположены по возрастанию.
\end{teacherbox}

Из цифр \(1,\ 3,\ 6\), двузначные без повторений:
\[
  13,\ 16;\qquad 31,\ 36;\qquad 61,\ 63.
\]

\begin{teacherbox}[Критерии упорядоченного списка]
  \begin{itemize}
    \item группы первых цифр идут по возрастанию;
    \item внутри группы следующие цифры перебираются по возрастанию;
    \item каждый допустимый выбор появляется один раз;
    \item количество полученных чисел совпадает с ожидаемым.
  \end{itemize}
\end{teacherbox}

\section{Ноль в начале записи}

\begin{teacherbox}[Точная формулировка]
  Запись \(038\) обозначает число \(38\), поэтому не является записью
  трёхзначного числа. Говорите «запись многозначного числа не начинается с
  нуля», а не «ноль не занимает разряд».
\end{teacherbox}

Для цифр \(0,\ 3,\ 8\): \(308,\ 380,\ 803,\ 830\).

\section{Проверка и сравнение}

\begin{teacherbox}[Один набор, два условия]
  Без повторений из \(1,\ 4,\ 7\) получаем 6 чисел:
  \(14,\ 17,\ 41,\ 47,\ 71,\ 74\).

  С повторениями получаем 9 чисел:
  \(11,\ 14,\ 17,\ 41,\ 44,\ 47,\ 71,\ 74,\ 77\).
  Добавились \(11,\ 44,\ 77\).
\end{teacherbox}

\begin{teacherbox}[Чужие списки]
  \begin{itemize}
    \item В \(13,\ 16,\ 31,\ 36,\ 61,\ 61\) число \(61\) повторено, число
      \(63\) пропущено.
    \item Из \(02,\ 06,\ 20,\ 26,\ 60,\ 62\) исключаются \(02\) и \(06\);
      остаются \(20,\ 26,\ 60,\ 62\).
  \end{itemize}
\end{teacherbox}

Условие для \(33,\ 35,\ 53,\ 55\): составить все двузначные числа, используя
только цифры \(3\) и \(5\); повторения разрешены, использовать обе цифры
необязательно.
При запрете повторений: \(35,\ 53\).

\section{Контрольная точка и маршрут}

\begin{teacherbox}[Ответы]
  \begin{enumerate}
    \item \(11,\ 14,\ 41,\ 44\);
    \item \(407,\ 470,\ 704,\ 740\);
    \item \(229,\ 292,\ 299,\ 922,\ 929,\ 992\);
    \item повторено \(72\), пропущено \(75\).
  \end{enumerate}
\end{teacherbox}

\begin{teacherbox}[Ожидаемое объяснение]
  В первом списке проверены первые цифры \(1\) и \(4\), а на второй позиции
  разрешены обе цифры. Во втором списке допустимы только первые цифры \(4\) и
  \(7\); после их выбора две оставшиеся цифры можно поставить двумя способами.
  В третьем списке исключены \(222\) и \(999\), потому что обе цифры
  обязательны. Списки сгруппированы и упорядочены по первой цифре.
\end{teacherbox}

\begin{resultbox}[Решение о переходе]
  \begin{itemize}
    \item \textbf{Переход к M06:} условия различены, списки полны и
      упорядочены, ноль не поставлен первым, пропуск и повтор найдены.
    \item \textbf{M05-R:} две или более ошибки либо повторяется случайный
      перебор, изменение условия или запись с нулём в начале.
    \item \textbf{Дополнительная практика или M05-H:} способ понятен, но
      ученику ещё нужна тренировка полноты и упорядочивания списка.
    \item \textbf{M05\(\star\):} основной результат достигнут устойчиво;
      лист можно выполнить дополнительно.
  \end{itemize}
\end{resultbox}

\begin{teacherbox}[Повторная проверка после M05-R]
  Из цифр \(0,\ 4,\ 8\) составить все двузначные числа без повторений:
  \(40,\ 48,\ 80,\ 84\). Затем из цифр \(2,\ 5\) составить трёхзначные числа
  с повторениями, если обе цифры обязательны:
  \(225,\ 252,\ 255,\ 522,\ 525,\ 552\).
\end{teacherbox}

\section{Ответы к дополнительным листам}

\begin{teacherbox}[Ответы M05-R]
  Дерево: \(30,\ 36,\ 60,\ 63\). С повторениями из \(3\) и \(7\):
  \(33,\ 37,\ 73,\ 77\); без повторений: \(37,\ 73\); при обязательности обеих
  цифр в трёхзначном числе:
  \(337,\ 373,\ 377,\ 733,\ 737,\ 773\).

  В списке \(13,\ 16,\ 31,\ 36,\ 61,\ 61\) повторено \(61\), пропущено \(63\).
  Повторная проверка: \(40,\ 48,\ 80,\ 84\) и
  \(225,\ 252,\ 255,\ 522,\ 525,\ 552\).
\end{teacherbox}

\begin{teacherbox}[Ответы M05\(\star\)]
  Список А полный: в каждой из трёх групп по первой цифре есть два порядка
  оставшихся цифр. Для \(0,\ 2,\ 5\) неполна группа с первой цифрой \(5\);
  пропущено \(520\).

  В списке \(13,\ 16,\ 31,\ 36,\ 61,\ 61\) одновременно есть повтор и пропуск.
  Для четырёх разных ненулевых цифр получается \(4\cdot3=12\) двузначных чисел
  без повторений: четыре группы по три числа. В последнем сравнении без
  повторений получается 6 чисел, а с повторениями \textemdash{} 9; во второй задаче
  на 3 числа больше, потому что добавляются три числа с одинаковыми цифрами.
\end{teacherbox}

\begin{notebox}[Домашняя работа]
  M05-H назначается после контрольной точки или после успешной повторной
  проверки M05-R. Требуйте не только списки, но и короткое объяснение их
  полноты.
\end{notebox}
