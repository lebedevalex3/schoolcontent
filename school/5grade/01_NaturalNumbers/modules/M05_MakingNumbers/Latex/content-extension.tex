\begin{warningbox}[1. Какой список полный]
  Из цифр \(1,\ 3,\ 8\) составляли трёхзначные числа без повторений, используя
  каждую цифру ровно один раз.

  Список А: \(138,\ 183,\ 318,\ 381,\ 813,\ 831\).

  Список Б: \(138,\ 183,\ 318,\ 381,\ 813\).

  Не перестраивая перебор заново, определите полный список и докажите ответ.
  \writinglines{3}
\end{warningbox}

\begin{practicebox}[5. Создайте проверяемый список]
  Выберите три разные ненулевые цифры. Составьте все трёхзначные числа,
  используя каждую цифру ровно один раз. Расположите числа группами по первой
  цифре.

  Цифры: \answerblank[40mm].

  Группа 1: \answerblank[72mm].

  Группа 2: \answerblank[72mm].

  Группа 3: \answerblank[72mm].

  Удалите одно число. Как другой ученик быстро обнаружит пропуск?
  \writinglines{2}
\end{practicebox}

\begin{checkpointbox}[6. Сравните без полного перебора]
  В одной задаче использованы три разные ненулевые цифры и составляются
  двузначные числа без повторений. В другой задаче повторения разрешены, а
  использовать все цифры необязательно.

  На сколько чисел во второй задаче больше? Объясните.
  \writinglines{3}
\end{checkpointbox}
