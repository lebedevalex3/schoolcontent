# Урок 3. Составление чисел. Задачи на цифры

Комплект материалов для 5 класса по теме «Обозначение натуральных чисел».
Это заключительный урок модуля из трёх занятий.

## Состав

- `content-reference.tex` — теоретический лист;
- `content-student.tex` — рабочий лист ученика;
- `content-teacher.tex` — версия учителя с ответами и комментариями;
- `homework-content.tex` — домашняя работа;
- `homework-solutions.tex` — решения и диагностические комментарии;
- `figures.tex` — TikZ-схемы;
- `style-common.sty`, `style-color.sty`, `style-bw.sty` — общие стили;
- `main-*.tex` — сборочные файлы;
- `pdf/` — готовые цветные и чёрно-белые PDF.

## Содержание урока

1. Порядок цифр и значение числа.
2. Составление всех чисел при разрешённом повторении цифр.
3. Составление чисел без повторений и запрет нуля в старшем разряде.
4. Наибольшее и наименьшее число из заданных цифр.
5. Упорядоченный перебор по условию на цифры.
6. Подсчёт количества подходящих чисел.

## Сборка

Требуется XeLaTeX и шрифты Tinos, Arimo, DejaVu Sans Mono.

```bash
bash build-all.sh
```
