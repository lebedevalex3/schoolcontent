# Лист 02. Длина отрезка. Измерение и построение

Комплект для первых уроков математики в 5 классе.

## Состав

- `content-reference.tex` — справочный теоретический лист;
- `content-student.tex` — рабочий лист ученика;
- `content-teacher.tex` — ответы и методические комментарии;
- `figures.tex` — TikZ-рисунки, в том числе шкалы линейки;
- `style-common.sty`, `style-color.sty`, `style-bw.sty` — стили;
- `main-*.tex` — точки сборки цветных и чёрно-белых версий.

## Содержание

- различие между отрезком и его длиной;
- правильное измерение линейкой;
- измерение при начале отсчёта не от нуля;
- единицы длины и их преобразование;
- построение отрезка заданной длины;
- равные отрезки;
- точка внутри отрезка и равенство `AB = AC + CB`.

## Сборка

Используйте XeLaTeX:

```bash
xelatex main-student-color.tex
xelatex main-teacher-color.tex
xelatex main-reference-color.tex
```
