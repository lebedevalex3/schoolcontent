# Словарь тегов

Дата создания: 2026-07-12.

**Правило:** новый тег сначала вписывается сюда, потом используется в `tags:` заметки. Это держит теги плоскими и согласованными между корнями.

## Ось: экзамен / курс

- `#ege` — задачи ЕГЭ
- `#oge` — задачи ОГЭ
- `#school-circle` — школьный кружок
- `#olympiad` — олимпиадные задачи

## Ось: предмет

- `#math` — общий тег для всего репозитория
- `#geometry` — геометрия (любая)
- `#algebra` — алгебра
- `#analysis` — математический анализ
- `#combinatorics` — комбинаторика

## Ось: подраздел предмета

- `#planimetry` — планиметрия
- `#stereometry` — стереометрия
- `#parametr` — задачи с параметром
- `#equation` — уравнения
- `#inequality` — неравенства
- `#function` — исследование функций

## Ось: тема

Свободный набор, но **каждый новый тег — сюда с коротким пояснением**.

- `#trapezoid` — трапеция
- `#midline` — средняя линия
- `#circumcenter` — описанная окружность / центр
- `#inscribed-angle` — вписанный угол
- `#modulus` — модуль
- `#locus` — геометрическое место точек (ГМТ)
- `#perpendicular-bisector` — серединный перпендикуляр
- `#auxiliary-construction` — вспомогательная конструкция

## Ось: сложность

- `#basic` — базовая
- `#medium` — средняя
- `#advanced` — повышенная
- `#olympiad-level` — олимпиадная

## Ось: роль в уроке

- `#intro` — вводная
- `#practice` — отработка
- `#synthesis` — обобщение
- `#diagnostic` — диагностика/контроль

---

## Что НЕ тег

- `type` (task / concept / lesson / ...) и `status` (draft / ready / ...) живут в frontmatter. Не дублируй в тегах.
- Конкретные ID (`TASK-PLAN-TRAP-CIRC-001`) — это не теги, это идентификаторы. В `tags:` не клади.
- Длинные описания через `/` (`math/geometry/planimetry`) — не используем. Словарь плоский.

## Процедура добавления нового тега

1. Подумай: точно нужен новый? Может, существующий + контекст в тексте заметки?
2. Если точно нужен — выбери ось выше.
3. Добавь запись сюда с однострочным пояснением.
4. Используй в заметке.
